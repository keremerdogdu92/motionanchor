# Mask Fixture Set

This directory is the shared input for mask-engine comparisons.

Each real fixture must include:

- source image or extracted frame,
- manually approved binary ground-truth mask,
- fixture metadata in `manifest.json`,
- feature tags such as `hair`, `cape`, `sword`, `staff`, `glow`, or `particles`,
- license/provenance notes confirming the asset may be used in the repository.

Do not commit provider secrets, private customer media, or unapproved third-party assets.
